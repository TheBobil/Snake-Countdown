One-time setup (~10 minutes):
  1. Go to console.firebase.google.com and sign in with a Google account.
  2. Click Add project, give it any name, skip Analytics.
  3. In the left menu: Build → Realtime Database → Create database. Pick any region. Start in test mode.
  4. Go to Project Settings (⚙️ top-left) → Your apps → click the </> Web icon → register the app.
  5. Copy the firebaseConfig object Firebase shows you.
  6. Open index.html in any text editor. Find the block near the top of the <script> labelled FIREBASE CONFIG.
  7. Replace the placeholder values with your copied config. Save the file.
  8. Push to GitHub — done! Every device will now share the same start date.

Note on password: The admin password is saved per-device in the browser. Set it on each device after first login, or just use the default snake123 and change it to the same thing on both your phones.
